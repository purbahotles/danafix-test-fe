## Installation

- Clone the repo
  ```sh
  git clone https://github.com/purbahotles/danafix-test-fe.git
  ```
- And then we can use the normal build and run procedure
  ```sh
  cd advance-material-table
  npm install
  npm start
  npm build
  ```
